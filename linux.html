Linux personalizado

Utilizaron Linux From Scratch ? Es genial, era un sueño que tenia de concretar, y tras mucho tiempo pude completarlo, esta muy muy bueno y lo recomiendo a todo Sysadmin que quiera aprender mas. El problema por el que no lo continue es que luego el mantenimiento es muy complicado, con una comunidad detras se hace mas facil y devertido obviamente. Por ese mismo motivo quedo en la experiencia.
Aunque, lo bueno es que me dio todo el conocimiento necesario para personalizar otras versiones, y asi pude concretar otro proyecto que tenia en mente: una distro de linux booteable de USB para una compu sin disco rígido para usar como Firewall.
Otro proyecto similar es una distro con zoom, no pude encontrar una y asi puedo bootear cualquier computadora con el pendrive y tener zoom sin necesidad de instalarlo. Facil y práctico, portatil para tener conferencias donde sea cuando sea !
